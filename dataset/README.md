## Prepare datasets

It is recommended to dataset root to `./dataset`.  
If your folder structure is different, you may need to change the corresponding paths in files.

```
.
├── dataset
    ├── train
    │   ├── close
    │   │   ├── image1.jpg
    │   │   ├── image2.jpg
    │   │   └── .....
    │   └── open
    ├── val
    │   ├── close
    │   └── open
    └── test
        ├── close
        └── open
